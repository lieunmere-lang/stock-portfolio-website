# Phase 3: 뉴스 리포트 UI 개선 — 설계 문서

## 1. 개요

매일 생성되는 뉴스 리포트를 효과적으로 탐색·소비할 수 있도록 UI를 개선한다.
현재는 최신 리포트 1건 보기 + 이전/다음 버튼만 있는 상태이며, 리포트가 쌓이면 탐색이 불편하다.

## 2. 변경 범위

| 항목 | 설명 |
|------|------|
| 리포트 목록 (리스트 뷰) | `news.html`에 토글로 리스트 뷰 추가 |
| 대시보드 미리보기 | `index.html`에 오늘 리포트 요약 카드 추가 |
| 빈 날짜 처리 | 리포트 있는 날짜만 이동 (건너뛰기) |
| 페이지네이션 | 더보기 버튼 (10건씩) |

## 3. `news.html` — 리스트 뷰 추가

### 3-1. 뷰 전환

헤더 영역에 토글 버튼 2개를 추가한다:
- **[리포트]** — 현재와 동일한 단일 리포트 상세 뷰 (기본값)
- **[목록]** — 리포트 리스트 뷰

뷰 전환 시 해당 영역만 show/hide 한다. (페이지 새로고침 없음)

### 3-2. 리스트 뷰 구조

```
┌─────────────────────────────────────────┐
│ 📊 뉴스 리포트        [리포트] [목록]    │
│                                         │
│ ┌─ 2026-04-12 ──────────────────────┐  │
│ │ 수집 287건 → 선별 12건 · Haiku    │  │
│ │ 미 연준이 금리 동결을 시사하며...   │  │
│ └───────────────────────────────────┘  │
│                                         │
│ ┌─ 2026-04-11 ──────────────────────┐  │
│ │ 수집 263건 → 선별 11건 · Haiku    │  │
│ │ 미국 3월 PPI가 예상치를 하회...     │  │
│ └───────────────────────────────────┘  │
│                                         │
│ ...10건 표시...                          │
│                                         │
│         [ 더보기 ]                       │
└─────────────────────────────────────────┘
```

- 각 카드에 표시: 날짜, 수집/선별 건수, 모델명, 요약 미리보기 (1줄 말줄임)
- 카드 클릭 → 리포트 뷰로 전환 + 해당 날짜 리포트 로드
- 처음 10건 표시, 더보기 버튼으로 10건씩 추가 로드
- 더 이상 데이터 없으면 더보기 버튼 숨김

### 3-3. 이전/다음 버튼 수정

- 리포트 뷰에서만 표시 (리스트 뷰에서는 숨김)
- 리포트 목록(`/api/news/reports`) 기반으로 이동하므로 리포트 있는 날짜만 이동
- 빈 날짜(주말/공휴일)는 자동으로 건너뜀

## 4. `index.html` — 대시보드 뉴스 미리보기 카드

### 4-1. 위치

대시보드 상단 요약 카드들 아래, 보유현황 테이블 위에 배치한다.

### 4-2. 카드 구조

```
┌─ 📊 오늘의 모닝 리포트 (2026-04-12) ──── [자세히 보기 →] ─┐
│                                                             │
│ 미 연준이 금리 동결을 시사하며 글로벌 증시가 상승세를       │
│ 보이고 있습니다. 비트코인 ETF로의 자금 유입이 지속...       │
│                                                             │
│ 주요 뉴스                                                   │
│ ★★★★★ 파월 의장, 금리 동결 시사 — '데이터 의존적'...  TSLA │
│ ★★★★★ RKLB, NASA 달 탐사 발사 계약 4억 달러 수주      RKLB │
│ ★★★★☆ 비트코인 현물 ETF, 주간 순유입 22억 달러             │
│                                                             │
│ 수집 287건 → 선별 12건                                      │
└─────────────────────────────────────────────────────────────┘
```

### 4-3. 동작

- `/api/news/latest` API 호출
- AI 요약 텍스트 표시 (2줄 말줄임 처리)
- 중요도(importance) 상위 2~3개 뉴스 제목 + 티커 뱃지 표시
- "자세히 보기" 클릭 → `/news.html`로 이동
- 리포트가 없으면 카드 자체를 숨김 (빈 공간 없음)

## 5. 백엔드 — 페이지네이션

### 5-1. `/api/news/reports` 변경

기존:
```
GET /api/news/reports → [ ... 전체 목록 ... ]
```

변경:
```
GET /api/news/reports?offset=0&limit=10
```

응답:
```json
{
  "items": [
    {
      "report_date": "2026-04-12",
      "created_at": "2026-04-12T09:00:00",
      "model_used": "claude-haiku-4-5",
      "total_collected": 287,
      "total_selected": 12,
      "summary_preview": "미 연준이 금리 동결을 시사하며..."
    }
  ],
  "total": 45,
  "has_more": true
}
```

- `offset`: 시작 위치 (기본값 0)
- `limit`: 조회 건수 (기본값 10)
- `total`: 전체 리포트 수
- `has_more`: 추가 데이터 존재 여부

### 5-2. 기존 엔드포인트 유지

| 엔드포인트 | 변경 |
|------------|------|
| `GET /api/news/latest` | 변경 없음 |
| `GET /api/news/report/{date}` | 변경 없음 |
| `GET /api/news/reports` | offset/limit 추가, 응답 형식 변경 |

## 6. 변경 파일

| 파일 | 변경 내용 |
|------|-----------|

| `frontend/index.html` | 뉴스 미리보기 카드 섹션 + JS 로직 추가 |
