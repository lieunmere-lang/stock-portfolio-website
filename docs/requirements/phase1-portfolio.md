# Phase 1: 포트폴리오 통계 및 데이터 분석

> 구현 완료 기준일: 2026-04-11

---

## 1-1. 대시보드 요약 카드 (상단)

| 카드 | 내용 |
|------|------|
| 총 평가금액 | 전체 자산 합계 (KRW) |
| 총 손익 | 금액 + 수익률 |
| 오늘의 손익 | 24시간 전 스냅샷 대비 변동 |
| 투자 원금 | 총 매수 금액 (avg_price × 수량 합계) |
| 포트폴리오 베타 | BTC 대비 변동성 지수 (CoinGecko, 비동기 로드) |

## 1-2. 보유 현황 테이블

보유 종목 전체를 한눈에 볼 수 있는 테이블 (1분마다 자동 갱신)

| 컬럼 | 내용 |
|------|------|
| 종목 | 이름 + 티커 |
| 매수가 | 평균 매수 단가 (KRW) |
| 현재가 | 실시간 현재가 (KRW) |
| 수량 | 보유 수량 |
| 매수금액 | avg_price × 수량 |
| 평가금액 | 현재가 × 수량 |
| 손익 | 평가금액 - 매수금액 + 수익률 % |
| 비중 | 전체 포트폴리오 대비 미니 바 |
| 보유기간 | 최초 매수일부터 경과 일수 |

- 손익 양수/음수 색상 구분 (초록/빨강)
- 업비트 Open API (`/v1/accounts`) 자동 연동
- **수동 자산 등록** (`manual_assets` DB 테이블): 업비트 API로 조회 불가한 자산(스테이킹 대기 등)을 수동 등록해 포트폴리오에 포함
  - 현재 등록: 이더리움 (스테이킹) — 6 ETH, 매입가 3,900,210원, 2023-03-01
  - ETH2 스테이킹 완료 시 `is_active=False` 처리 후 업비트 자동 조회로 전환

## 1-3. 수익률 분석

- **기간별 수익률**: 1일, 1주, 1개월, 3개월, 6개월, 1년 (DB 스냅샷 기반)
- **누적 수익률 차트**: 기간 선택(1개월/3개월/6개월/1년) 라인 차트 (Chart.js)
  - 툴팁: 날짜 + 수익률(%) + 총 평가금액(KRW) 동시 표시
  - x축 레이블: 기간에 따라 일/월 단위 자동 전환
- **최대 낙폭(MDD)**: 고점 대비 최대 하락률 (1년 기준)
- **변동성**: 일별 수익률의 연환산 표준편차 (30일 기준)
- **샤프 지수**: (연환산 수익률 - 무위험이자율 3.5%) / 연환산 변동성

### 과거 데이터 백필

- 서버 최초 기동 시 업비트 일봉 캔들 API(`/v1/candles/days`, 인증 불필요)로 최대 365일치 가격 데이터 자동 수집
- 현재 보유 수량 기준으로 날짜별 포트폴리오 가치를 역산해 DB에 삽입 (시뮬레이션 데이터)
- 이미 스냅샷이 있는 날짜는 스킵 (중복 방지)
- 수동 자산(ETH2)도 백필 시 ETH 캔들 가격 기준으로 포함

## 1-4. 자산 배분 분석

- **종목별 비중**: 도넛 차트
- **트리맵**: 비중 크기 시각화, 색상으로 수익률 구분
- **상위 5 종목**: 수익 기여도 순 막대 그래프

## 1-5. 리스크 지표

- **집중도 위험**: HHI(허핀달-허슈만 지수) — Σ(비중²) × 10000, 2500 이상 경고
- **상관관계 매트릭스**: 보유 코인 간 가격 움직임 상관도 히트맵 (CoinGecko 기반, 수동 로드)

## 1-6. 데이터 동기화

- **자동 동기화**: 1분 간격 APScheduler (서버 시작 즉시 첫 실행)
- **프론트엔드 자동 갱신**: 60초마다 전체 대시보드 데이터 재조회
- 수동 동기화 버튼 제거 → 네비게이션 바에 마지막 업데이트 시각 표시
