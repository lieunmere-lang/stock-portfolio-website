| 항목 | 설명 |
|------|------|

## 2-2. 보유 주식 워치리스트 (시장 데이터)

보유 주식의 시장 데이터 테이블 (30초~1분 주기 자동 갱신)

| 컬럼 | 내용 | 비고 |
|------|------|------|
| Symbol | 티커 | |
| Price | 현재가 + 시간외 가격 | 장 마감 후 after-hours 표시 |
| Change / Change % | 당일 등락 (금액 + %) | 색상으로 +/- 구분 |
| Volume / Avg Vol | 당일 거래량 / 평균 거래량 | 과열 여부 판단 |
| Day Range | 당일 고/저 | 미니 바 차트로 현재가 위치 표시 |
| 52W Range | 52주 고/저 | 미니 바 차트로 현재가 위치 표시 |
| Analyst Rating | Wall Street 컨센서스 요약 | Buy / Hold / Sell 아이콘 |

**데이터 소스:** `yfinance` (15분 지연, 무료) — 개인용으로 충분

> 진짜 실시간(0초 지연)이 필요하면 Polygon.io 무료 티어 고려 가능

## 2-3. 주식 상세 정보 (클릭 시 모달) — 네이버 금융 + Yahoo Finance

| 항목 | 설명 |
|------|------|
| PER | 주가수익비율 (Price/Earnings) |
| PBR | 주가순자산비율 (Price/Book) |
| PSR | 주가매출비율 (Price/Sales) |
| EV/EBITDA | 기업가치 대비 영업이익 |
| ROE / ROA | 자기자본·총자산 수익률 |
| 배당수익률 | 연간 배당 / 현재가 |
| 실적 발표일 | 다음 어닝 날짜 (Earnings Date) |
| 시장 컨센서스 | 애널리스트 목표가 + 매수/중립/매도 비율 |
| 52주 고/저 | 연간 최고·최저가 |
| 외인·기관 순매수 | 최근 수급 동향 |
