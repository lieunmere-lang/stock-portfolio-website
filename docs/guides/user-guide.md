# 포트폴리오 대시보드 사용 가이드

> 코인·주식 통합 포트폴리오 대시보드 이용 방법

---

## 접속 방법

### 집 Wi-Fi (PC / 핸드폰 공통)

맥북과 같은 Wi-Fi에 연결되어 있다면 로컬 IP로 직접 접속합니다.

```
http://192.168.x.x:8000
```

> 맥북 IP는 서버 관리자에게 문의하거나, 맥북에서 `ipconfig getifaddr en0` 명령으로 확인합니다.

### 외부 / 핸드폰 (ngrok 터널)

서버 관리자에게 현재 URL을 받아 접속합니다.

```
https://xxxx-xxxx.ngrok-free.app
```

- 처음 접속 시 **"You are about to visit..."** 경고 페이지가 뜨면 **"Visit Site"** 버튼을 누릅니다.
- 무료 ngrok은 서버 재시작마다 URL이 바뀌므로, 변경 시 서버 관리자에게 새 URL을 받으세요.

## 로그인

| 항목 | 값 |
|------|----|
| 아이디 | 서버 관리자가 설정한 ID |
| 비밀번호 | 서버 관리자가 설정한 PW |

로그인 후 토큰이 브라우저에 저장되어 **30일간 자동 로그인**됩니다.

## 대시보드 구성

### 상단 요약 카드

| 카드 | 설명 |
|------|------|
| 총 평가금액 | 현재 보유 자산의 시세 합계 |
| 총 손익 | 매수 원금 대비 수익/손실 금액 및 수익률 |
| 오늘의 손익 | 24시간 전 대비 변동 금액 |
| 투자 원금 | 매수에 투입한 총 금액 |
| 포트폴리오 베타 | BTC 시장 대비 변동성 (1.0 = 시장과 동일) |

### 수익률 분석

- **기간별 수익률**: 1일 / 1주 / 1개월 / 3개월 / 6개월 / 1년 기준 수익률
- **누적 수익률 차트**: 우측 버튼으로 기간 변경 가능 (1개월 / 3개월 / 6개월 / 1년)

### 리스크 지표

| 지표 | 설명 |
|------|------|
| 최대 낙폭 (MDD) | 고점 대비 최대 하락폭. 클수록 위험 |
| 변동성 | 연환산 일별 수익률 표준편차. 클수록 가격 변동이 큼 |
| 샤프 지수 | 위험 대비 수익 효율. 1 이상이면 양호 |
| HHI (집중도) | 특정 종목에 자산이 몰려있는 정도. 2500 이상이면 집중 위험 |

### 보유 현황 테이블

보유 중인 모든 종목을 한눈에 확인합니다.

| 컬럼 | 설명 |
|------|------|
| 종목 | 이름 및 티커 |
| 매수가 | 평균 매수 단가 |
| 현재가 | 실시간 현재 가격 |
| 수량 | 보유 수량 |
| 매수금액 | 평균 매수가 × 수량 |
| 평가금액 | 현재가 × 수량 |
| 손익 | 평가금액 - 매수금액 |
| 수익률 | 손익 / 매수금액 (%) |
| 비중 | 전체 포트폴리오 중 해당 종목 비율 |
| 보유기간 | 최초 매수일부터 경과 기간 |

### 자산 배분

- **도넛 차트**: 종목별 비중을 시각적으로 확인
- **트리맵**: 비중이 클수록 크게 표시, 색상으로 수익률 구분

### 수익 기여도 Top 5

손익 기준 상위 5개 종목과 기여 비율을 막대 그래프로 표시합니다.

### 상관관계 매트릭스

- 보유 코인 간 가격 움직임의 상관도를 히트맵으로 표시
- **빨간색**: 함께 오르내리는 경향 (양의 상관)
- **파란색**: 반대로 움직이는 경향 (음의 상관)
- **"불러오기" 버튼**을 눌러야 CoinGecko에서 데이터를 받아옵니다 (처음 1~2초 소요)

## 데이터 동기화

- **자동 동기화**: 1시간마다 업비트 최신 데이터를 자동으로 갱신합니다.
- **수동 동기화**: 네비게이션 바 우측의 **"동기화"** 버튼을 누르면 즉시 갱신됩니다.

---

## 로그아웃

네비게이션 바 우측의 **"로그아웃"** 버튼을 누릅니다.  
로그아웃 후에는 다시 로그인 페이지로 이동합니다.

## 자주 묻는 질문

**Q. "데이터 로딩 실패" 메시지가 뜹니다.**  
→ 서버가 꺼져 있습니다. 서버 관리자에게 서버 시작을 요청하세요.

**Q. 수익률이 0% 또는 "데이터 부족"으로 뜹니다.**  
→ DB에 충분한 스냅샷이 쌓이지 않은 경우입니다. 1일 이상 서버가 운영되면 자동으로 표시됩니다.

**Q. 베타 / 상관관계가 "데이터 부족"으로 뜹니다.**  
→ CoinGecko API에 등록되지 않은 코인이거나, API 호출에 실패한 경우입니다. 잠시 후 재시도해 주세요.

**Q. ngrok URL로 접속이 안 됩니다.**  
→ URL이 바뀌었을 수 있습니다. 서버 관리자에게 최신 URL을 요청하세요.
